GreenCart Inventory Query API

Read-only access to GreenCart's live supermarket inventory. Query stock levels, search products, browse categories, and check availability by store — all without an account.

 v1.3.0
 Read-only
 Rate limit: 120 req / min
 Stock refreshed: every 60 s
Server
https://inventory.greencart.onrender.com/v1 Production

This API is fully public. No API key, token, or account is needed. All endpoints are read-only — no writes, no deletions. Rate limiting applies per IP address.

Inventory

4 endpoints
GET /inventory Browse all stock

Returns current stock levels across all GreenCart stores. Filter by store, category, or availability. Paginated, 50 items per page by default.

Query parameters

NameTypeDescription
store_idstringFilter to one store
categorystringproduce, dairy, bakery, frozen…
in_stockbooleanOnly show available items
pageintegerPage number (default 1)
limitintegerPer page, max 100

Responses

200Paginated array of InventoryItem objects
400Invalid query parameter
{ "data": [{ "sku": "PRD-BAN-ORG", "name": "Organic Bananas", "store_id": "store_001", "qty_available": 214, "unit": "lb", "price": 0.59, "last_updated": "2026-07-14T09:00:00Z" }], "page": 1, "total": 3841 }
GET /inventory/{sku} Stock for one product

Returns stock levels for a single SKU across all stores, or filtered to one store via query param.

Path parameters

NameTypeDescription
sku *stringProduct SKU (e.g. PRD-BAN-ORG)

Query parameters

NameTypeDescription
store_idstringScope to a specific store

Responses

200Array of StoreStock objects for this SKU
404SKU not found
[{ "store_id": "store_001", "store_name": "GreenCart Downtown", "qty_available": 214, "low_stock": false }]
GET /inventory/low-stock Items running low

Returns all items currently at or below the public restock threshold (20 units or fewer). Useful for shoppers checking if an item will still be available.

Query parameters

NameTypeDescription
store_idstringScope to one store
categorystringFilter by category

Responses

200Array of LowStockItem objects
GET /inventory/search Search by name or SKU

Full-text search across product names, SKUs, and descriptions. Returns matching items with stock levels for the requested store.

Query parameters

NameTypeDescription
q *stringSearch term (min 2 characters)
store_idstringScope results to a store
categorystringNarrow to a category

Responses

200Array of SearchResult objects
400Query too short or missing
GET /inventory/search?q=oat+milk&store_id=store_003 { "results": [{ "sku": "PRD-OATM-PL", "name": "Oat Milk Plain 32oz", "qty_available": 48, "price": 4.29 }], "count": 6 }

Products

3 endpoints
GET /products Full product catalog

Browse the complete GreenCart product catalog. Includes name, SKU, category, unit, price, and nutritional flags. Does not include live stock — pair with /inventory for availability.

Query parameters

NameTypeDescription
categorystringFilter by category
brandstringFilter by brand name
organicbooleanOnly certified organic items
pageintegerPage number (default 1)

Responses

200Paginated Product array
GET /products/{sku} Single product detail

Full product record including description, allergens, nutritional info, weight, and brand details.

Responses

200Full Product object
404SKU not found
{ "sku": "PRD-BAN-ORG", "name": "Organic Bananas", "brand": "FarmFresh", "category": "produce", "unit": "lb", "price": 0.59, "organic": true, "allergens": [] }
GET /products/{sku}/availability Product and stock combined

Convenience endpoint that returns product details and live stock levels across all stores in one response. Avoids a second /inventory call.

Responses

200Product object with embedded store_stock array
404SKU not found
